# MERN Stack Application

## Installation

Use the package manager [npm | yarn] install dependencies


```bash
git clone project
```

```bash
cd projectDirectory && cd backend
```

```bash
npm install | yarn install
```

Start mongodb service

```bash
mongod
```

```bash
npm start 
```

Application will be started at localhost:3000




## Stack

* Mongodb
* Node.js
* Express
* React
* Redux
